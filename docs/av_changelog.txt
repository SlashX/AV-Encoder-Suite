  ADD Triple-layer encoding: DV Profile 8.1 + HDR10 + HDR10+.
  Un singur fisier care contine simultan Dolby Vision, HDR10 static
  si HDR10+ metadata dinamica. Orice TV primeste cea mai buna
  experienta pe care o suporta (DV > HDR10+ > HDR10 > SDR).

    1) hdr10plus_tool extract → extrage HDR10+ din sursa → JSON

  v30.1 — Eliminare IAMF (nu este codec real)
═══════════════════════════════════════════════════════════════

  [REMOVE] Optiunea IAMF eliminata din toate meniurile audio.
  IAMF (Immersive Audio Model and Formats) nu este un codec audio —
  este un format/container spatial. Optiunea facea exact acelasi lucru
  ca Opus (optiunea 3) sau FLAC (optiunea 5), fara metadata spatiala.
  ffmpeg 8.1 are doar muxer/demuxer raw .iamf, nu generare metadata.

  Eliminat din: av_launcher.sh (optiunea 9 → copy revine la 9),
  av_common.sh (get_audio_params iamf case + interactive dialog),
  av_encoder_audio.sh (optiunea 9 + build params + IAMF+mov warning),
  av_encode.ps1 (ambele meniuri + encode loop + audio-only + profile),
  av_info.txt (optiune audio + nota IAMF + container compat).

  Audio: 9 optiuni (1-AAC, 2-AAC custom, 3-Opus, 4-Opus custom,
  5-FLAC, 6-FLAC custom, 7-E-AC3, 8-LPCM, 9-Copy).
